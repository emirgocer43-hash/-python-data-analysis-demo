# Python Data Analysis & Plotting Tool

This project provides a simple Python script that analyzes CSV data and
automatically generates clean, publication-ready plots.

## What this script does
- Reads CSV and Excel-style numerical data
- Generates clear and labeled plots
- Saves plots as image files
- Works for academic and student projects

## Typical use cases
- Physics and engineering lab data
- University assignments
- Quick data visualization for reports

## How to use
1. Place your CSV file in the project folder
2. Run `analyze_data.py`
3. Get your plot instantly as an image

## Technologies
- Python
- Pandas
- Matplotlib
